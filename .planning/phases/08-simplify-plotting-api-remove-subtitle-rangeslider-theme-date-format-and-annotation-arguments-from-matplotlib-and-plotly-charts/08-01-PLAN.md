# Plan 08-01: Remove subtitle, rangeslider, theme, and annotations from plots.py and __init__.py

## Objective

Simplify `plots.py` by removing five groups of arguments from all function signatures,
docstrings, internal logic, and validation code. Remove the `add_annotation()` public
function entirely and clean up `__init__.py`. The goal is substantially simpler code
with no dead parameters.

## Context

@src/pxts/plots.py
@src/pxts/__init__.py

## Removals summary

| Argument | Remove from |
|---|---|
| `subtitle` | public API (`tsplot`, `tsplot_dual`), `_validate_plot_params`, `_plot_ts_mpl`, `_plot_ts_plotly`, `_plot_ts_dual_mpl`, `_plot_ts_dual_plotly` |
| `rangeslider` | public API (`tsplot`, `tsplot_dual`), `_plot_ts_plotly`, `_plot_ts_dual_plotly` |
| `theme` | public API (`tsplot`, `tsplot_dual`), `_plot_ts_plotly`, `_plot_ts_dual_plotly` |
| `date_format` | `_plot_ts_mpl`, `_plot_ts_dual_mpl` only (keep in public API + Plotly renderers) |
| `annotations` + `add_annotation()` | public API (`tsplot`, `tsplot_dual`), `_validate_plot_params`, `_plot_ts_plotly`, `_plot_ts_dual_plotly`, the `add_annotation()` function body, `__init__.py` import + `__all__` |
