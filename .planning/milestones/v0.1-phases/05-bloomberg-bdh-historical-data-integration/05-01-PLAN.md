Add read_bdh() to src/pxts/io.py after write_ts(). The function must:

1. Import pdblp at call time (not module level) with ImportError fallback:
   ```python
   try:
       import pdblp
   except ImportError:
       raise ImportError(
           "pdblp required for read_bdh(). Install with: pip install pxts[bloomberg]"
       )
   ```

2. Accept signature: `read_bdh(tickers, start, field='PX_LAST', end=None)`
   - `tickers`: list of Bloomberg ticker strings (e.g., ['AAPL US Equity'])
   - `start`: str, datetime, or pd.Timestamp — convert to 'YYYYMMDD' string via pd.to_datetime(start).strftime('%Y%m%d')
   - `field`: single string, default 'PX_LAST'
   - `end`: optional, defaults to pd.Timestamp.today() when None — also converted to 'YYYYMMDD'

3. Open, fetch, close BCon using pdblp.BCon(port=8194, timeout=20):
   ```python
   con = pdblp.BCon(port=8194, timeout=20)
   con.start()
   try:
       raw = con.bdh(tickers, field, start_date, end_date)
   finally:
       con.stop()
   ```

4. Reshape raw output to wide format:
   - pdblp.BCon.bdh() returns a DataFrame with a MultiIndex column (ticker, field)
   - For a single field, reshape to wide: `df = raw.xs(field, axis=1, level=1)` — this gives DatetimeIndex rows, one column per ticker
   - Column names become raw Bloomberg ticker strings

5. Call validate_ts(df) before returning.

6. Return df.

Write a module-level docstring update at the top of io.py adding read_bdh to the Public API list.

In pyproject.toml, add under [project.optional-dependencies]:
